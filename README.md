# FinFit – AI 금융 문해력 향상 서비스

“내 소비습관을 분석하고, 금융을 쉽게 설명하며, 실행 가능한 미션을 추천해주는 AI 금융 코치”

⸻

📌 1. 프로젝트 개요 (Overview)

<b>FinFit(핀핏)</b>은 청년층의 금융 문해력 하락 문제를 해결하기 위한
<b>AI 기반 금융 습관 분석 및 행동 교정 서비스</b>입니다.

사용자가 본인 거래내역 CSV 파일을 직접 업로드하면,
AI가 소비 패턴을 분석하고 금융 피드백 & 실천 미션을 추천합니다.
또한 공공데이터의 경제 용어를 하루 1개 제공하고, GPT API를 통해 <b>비유 기반 “쉬운 한 줄 설명”을 제공</b>하여 금융 개념 이해를 돕습니다.

📌 2. 주요 기능 (MVP Scope)
```
✔️ (1) 소비습관 분석 & 금융 피드백
	•	사용자 CSV 업로드
	•	거래내역 → 카테고리 자동 분류 (식비/교통/쇼핑/구독 등)
	•	월간 소비 요약, 고정비·변동비 비율
	•	“또래 대비 ○○% 높음” 등 단순 비교 피드백

✔️ (2) 실천 미션 추천
	•	과소비 카테고리에 따른 미션 추천
	•	예) 배달 1회 줄이기 / 구독 1개 정리하기
	•	절감 예상액 시각화

✔️ (3) 최신 경제 뉴스 제목 제공
	•	네이버 검색 API → 경제 뉴스 “제목만” 제공
	•	제목 클릭 → 언론사 원문 이동 (저작권 준수)

✔️ (4) 오늘의 경제용어
	•	공공데이터포털 “시사·경제 용어사전” 활용
	•	하루 1개 용어 랜덤 제공
	•	GPT API가 비유 기반 쉬운 한 줄 설명 생성
```

📌 4. 기술 스택
```
✔️ Backend (Spring Boot)
	•	Spring Boot 3.x
    •	Spring Web / Spring Data JPA
	•	Lombok
	•	PostgreSQL
	•	GPT API 연동
	•	CSV 파서(OpenCSV 또는 자바 기본 CSV 처리)

✔️ Frontend
	•	React
	•	TypeScript
	•	Tailwind CSS
	•	Chart.js / Recharts

✔️ AI 모델
	•	소비 카테고리 분류 → LightGBM 또는 BERT-small
	•	미션 추천 → Rule-based + Thompson Sampling
	•	경제용어 한줄 설명 → GPT API
	•	소비 패턴 분석 → 통계 기반
```

📌 6. MVP 개발 방법론 (핵심)

```
아래는 해커톤 제출용 MVP 개발을 위한 공식 방법론 문서 형태입니다.

🔧 Step 1. 데이터 확보 및 정제

(1) CSV 업로드 스펙 정의
	•	필수 필드: 날짜, 금액, 가맹점명, 메모
	•	CSV 샘플 제공

(2) 사용자가 업로드한 CSV → 서버 저장(X) → 메모리 기반 처리
	•	개인정보 최소화
	•	세션/일회성 분석만 수행

🔧 Step 2. 소비 카테고리 분류 모델 구축

방법 1 — Rule-based (MVP)
	•	가맹점 키워드 사전 구축
	•	예: “배민/요기요 → 배달”, “GS25/CU → 편의점”

방법 2 — ML 모델 병행(선택)
	•	LightGBM + TF-IDF
	•	또는 BERT-small fine-tuning

🔧 Step 3. 소비 패턴 분석 로직
	•	월간 소비 총액
	•	카테고리 비중
	•	고정비/변동비 자동 분류
	•	또래 평균 대비 비교(KOSIS 기준)
	•	이상 소비 탐지(Isolation Forest, 선택)

🔧 Step 4. 실천 미션 추천

MVP 방식
	•	룰 기반: 과소비 카테고리 → 미션 매핑
	•	미션 템플릿 제공
	•	절감액 추정 = 최근 지출 평균 × 미션 감축량

🔧 Step 5. 경제 뉴스 제목 제공
	•	네이버 검색 API 사용
	•	제목·언론사·URL만 사용자에게 노출
	•	기사 가공 금지(저작권 준수)

🔧 Step 6. 오늘의 경제 용어 + GPT 한줄 설명
	•	공공데이터 Excel 로컬 로드
	•	하루 1개 랜덤 용어 제공
	•	GPT API 프롬프트 예시:

"다음 경제 개념을 중학생도 알 수 있게 비유해서
한 문장으로 설명해줘: {용어: 설명}"

🔧 Step 7. 프론트엔드 UI 개발
	•	CSV 업로드 페이지
	•	대시보드(그래프/지표)
	•	미션 추천 카드
	•	경제 뉴스 목록
	•	오늘의 경제 용어(원문 + AI 한줄 설명)

🔧 Step 8. 테스트 / 배포
	•	로컬 CSV 파일 테스트
	•	GPT API Key 환경변수 처리
	•	Docker 또는 AWS EC2 + RDS 배포
```

📌 8. 라이선스 / 저작권 준수 가이드
	•	뉴스 본문 저장/요약 금지 → 제목만 제공, 원문 링크 연결
	•	공공데이터는 원문 그대로 제공
	•	GPT API는 공공데이터 원문을 비유 표현으로 재작성하는 용도만 사용
	•	사용자 CSV는 저장하지 않고 일회성 분석 처리
